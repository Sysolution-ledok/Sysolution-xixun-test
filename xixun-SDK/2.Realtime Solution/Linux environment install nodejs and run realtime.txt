一：执行命令  curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -  
二：执行命令  sudo apt-get install -y nodejs
三：执行命令 node -v。如果显示nodejs版本号，则安装成功
四：将realtime上传到Linux服务器上，进入到/RealtimeServer/bin目录下
五：执行命令node www，即可运行realtime



1. execute command curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -  
2. execute command sudo apt-get install -y nodejs
3. execute command node -v, if show nodejs version number means success
4. upload realtime to Linux server and enter to /RealtimeServer/bin directory
5. execute command node www then can run realtime
